
# 🤖 Chenura AI Telegram Bot https://t.me/Cheenura_ai_chat_bot

A professional Telegram AI chatbot powered by **Gemini 2.5 Flash**, built with:

- Flask
- MongoDB
- Telegram Bot API
- Google Gemini AI
- Admin Dashboard
- User Tracking
- Encrypted API Key Storage

---

# 🚀 Features

## 🤖 AI Chat
- Uses Gemini 2.5 Flash
- Users can chat with AI directly in Telegram

---

## 🔑 User API Key System
Users can:
- Add their own Gemini API key
- Store it securely using encryption

Command:
```bash
/setkey
````

---

## 👥 User Tracking

Tracks:

* Total users
* Usernames
* Usage count

MongoDB stores all user data.

---

## 📊 Admin Dashboard

Modern dashboard with:

* Dark UI
* Charts
* User statistics
* Top users table

Dashboard:

```bash
/login
```

---

## 🔐 Secure Login System

Protected admin panel using:

* Username
* Password
* Flask sessions

---

## 👑 Admin Commands

### View Total Users

Only admin can use:

```bash
/users
```

---

## 📈 Usage Tracking

Users can check usage:

```bash
/myusage
```
